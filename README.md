Thermostat-js
=============

A smart thermostat program, built entirely in javascript.

Technologies Used
-----------------
+ Javascript
+ JQuery
+ Git
+ Jasmine


### How to set it up
```sh
git clone git@github.com:aitkenster/thermostat-js.git
cd thermostat-js
```


How to run it:
### How to run it

Open index.html in the browser.

### How to test it:

Open /lib/jasmine/SpecRunner.html in the browser.
